# OpenAlcohol
Due to the COVID pandemic, a hand free, a simple, wall-attached alcohol dispenser was designed. 

<h3> Materials </h3>
<h6> Hardware</h6>
<ul>
  <li>Arduino Uno</li>
  <li>Solenoid Valve</li>
  <li>Ultrasonic Sensor</li>
  <li>Relay Switch</li>
  <li>Wires</li>
</ul>

<h6> Software</h6>
<ul>
  <li>Arduino IDE</li>
  <li>NewPing Library</li>
</ul>
